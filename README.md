# Crypto AML System Project

## Project Overview
Blockchain-based Anti-Money Laundering (AML) detection system for analyzing cryptocurrency transactions, detecting suspicious patterns, and visualizing transaction flows.

## Project Background
This repository contains the AML-focused implementation extracted from a larger collaborative blockchain intelligence project developed by a team.

Original collaborative repository:
https://github.com/nebu20/Crypto_AML_System

## Contribution
Contributed across multiple components of the blockchain AML system including:
- AML analytics workflows  
- Transaction monitoring modules  
- Clustering logic  
- System integration and development  

This was done as part of a collaborative team project.
